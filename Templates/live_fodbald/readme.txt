Instalation manual:
- Pull all code.
- Install rp_cms_steve_integration_live_fodbald module.
- Install live_fodbald theme.
- On this pageі admin/config/search/path/settings in Update action section please set the option “Create a new alias. Leave the existing alias functioning”

Check if rp_client_base is enabled. If no, please enable it.

To show teams on “team list” page You need to configure what You want to show.
For this use this page admin/structure/block/manage/fodbaldteamslistplugin.
In “Select League” field please choose nedeed league and in Select Teams field select needed teams for selected league. Save the block.
